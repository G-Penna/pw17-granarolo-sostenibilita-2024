# PW17 — Granarolo Sostenibilità 2024

Pagina web realizzata per Project Work universitario: sintesi didattica (non ufficiale) del Bilancio di Sostenibilità Granarolo 2024.

## Demo
https://g-penna.github.io/pw17-granarolo-sostenibilita-2024/

## Tecnologie
- HTML5
- CSS3 (responsive)
- JavaScript (interazioni base)

## Avvio in locale
Apri `index.html` con un browser oppure usa un server locale (es. VS Code Live Server).

- `index.html` — pagina principale
- `style.css` — stile
- `img/` — immagini
- `icons/` — icone
